===============================================================================
                           ANIMATION TIMINGS
===============================================================================

PLAYER ANIMATIONS:
- Idle: 1.0 second cycle (2 frames at 0.5s each)
- Walk: 0.4 second cycle (4 frames at 0.1s each)
- Jump: 0.2 second per frame (static poses)

COLLECTIBLE ANIMATIONS:
- Floating: 2.0 second cycle (smooth sine wave)
- Sparkle: 0.5 second burst effect
- Rotation: 3.0 second full rotation

ENEMY ANIMATIONS:
- Bear Walk: 0.8 second cycle (4 frames at 0.2s each)
- Bee Wings: 0.2 second cycle (4 frames at 0.05s each)
- Tornado Spin: 0.1 second cycle (8 frames at 0.0125s each)

ENVIRONMENT ANIMATIONS:
- Flag Wave: 1.5 second cycle (4 frames)
- Water Ripple: 2.0 second cycle (3 frames)
- Cloud Drift: 30 second full screen traverse

===============================================================================
                            IMPLEMENTATION NOTES
===============================================================================

SPRITE SHEETS: All animations stored in horizontal sprite sheets
TRANSPARENCY: Use PNG format with alpha channel
SCALING: Sprites designed for 2x or 3x scaling on high-DPI displays
COMPRESSION: Optimize PNG files for smaller file sizes
CONSISTENCY: Maintain consistent lighting direction (top-left)

FILE NAMING CONVENTION:
